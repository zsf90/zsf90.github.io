Rust 相关命令

发表于2022-04-23|更新于2022-04-23|Rust
|阅读量:

该篇文章记录了 Rust 相关的命令，以便不时之需。

1. rustc –version

查看 rust 的版本号

2. rustup update

更新 rust 版本

3. rustup doc

打开本地文档

4. 创建 Rust 项目

rust 项目的创建使用 cargo 工具生成管理的。

  • cargo new hello_rust 创建项目（可执行）
  • cargo build 编译项目
  • cargo check 检查项目
  • cargo run 运行项目
  • cargo build –release 以 Release 模式编译
  • cargo run –release 以 Release 模式运行

5. 查看命令帮助

cargo new –help，可以查看该命令相关的帮助。
